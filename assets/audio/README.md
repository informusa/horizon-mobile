# Game Audio Files

This directory contains sound effects for Horizon Mobile.

## Required Sound Files

Download these free sound effects from Pixabay and place them in this directory:

### 1. Jump Sound (`jump.mp3`)
- **URL**: https://pixabay.com/sound-effects/search/retro%20jump/
- **Recommended**: "Retro Jump Sound 01" or "8-bit Jump"
- **Usage**: Plays when player jumps

### 2. Collision Sound (`collision.mp3`)
- **URL**: https://pixabay.com/sound-effects/search/retro%20hit/
- **Recommended**: "Retro Hurt Sound" or "8-bit Hit"
- **Usage**: Plays when player hits a barrel

### 3. Power-up Sound (`powerup.mp3`)
- **URL**: https://pixabay.com/sound-effects/search/powerup/
- **Recommended**: "Power Up" or "Collect Coin"
- **Usage**: Plays when collecting power-ups

### 4. Game Over Sound (`gameover.mp3`)
- **URL**: https://pixabay.com/sound-effects/search/game%20over/
- **Recommended**: "Game Over" or "Retro Fail"
- **Usage**: Plays when game ends

### 5. Level Complete Sound (`levelcomplete.mp3`)
- **URL**: https://pixabay.com/sound-effects/search/level%20complete/
- **Recommended**: "Victory" or "Level Up"
- **Usage**: Plays when completing a level

### 6. Background Music (`background.mp3`)
- **URL**: https://pixabay.com/music/search/8-bit/
- **Recommended**: Any 8-bit or chiptune loop
- **Usage**: Plays during gameplay (looping)

## How to Download

1. Visit the Pixabay URL for each sound
2. Click on a sound effect you like
3. Click the green "Download" button
4. Save the file to this directory with the correct name

The audio manager is configured and ready. Once you add the MP3 files to this directory, they will automatically work in the game.
